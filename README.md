# AI 개인 금융 자동 설계 앱

## 개요
월수입 목표 입력 → 최적의 자산 배분 자동 설계
- 현금흐름 (정기예금/적금) | 투자 (ETF) | 부동산 (REIT) | 배당 (배당주)
- 상세 종목 추천 및 예상 수익/배당금 계산
- 반응형 웹 디자인

## 🌐 접속 정보

## 실행 방법

**백엔드:**
```bash
cd backend
pip install fastapi uvicorn python-multipart
uvicorn main:app --host 0.0.0.0 --port 8000
```

**프론트엔드:**
```bash
cd frontend
npm install
npm run dev -- --host
```

## 주요 기능
- 자산 배분 계산 (리스크 레벨별: 보수적/중립/공격적)
- 상세 종목 추천 (정기예금, ETF, 배당주, REIT)
- 예상 수익 & 배당금 계산
- 월별 예상 배당금 누적 시뮬레이션
- 반응형 UI (모바일/태블릿/PC)
